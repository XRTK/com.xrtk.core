Enum OculusApi.RawAxis1D

Oculus native single axis definitions

Namespace: XRTK.Oculus.Plugins
Assembly: XRTK.Oculus.dll
Syntax
[Flags]
public enum RawAxis1D
Remarks

Oculus API only uses RAW definitions. Oculus Asset also uses Virtual mappings, but it's not clear why (Oculus) Raw 1-dimensional axis (float) mappings that can be used to directly query the state of a controller.
