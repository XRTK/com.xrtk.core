Class MixedRealityToolkit

This class is responsible for coordinating the operation of the Mixed Reality Toolkit. It is the only Singleton in the entire project. It provides a service registry for all active services that are used within a project as well as providing the active configuration profile for the project. The ActiveProfile can be swapped out at any time to meet the needs of your project.
