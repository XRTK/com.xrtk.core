StartCommandLineBuild()

Start a build using Unity's command line. Valid arguments:

-autoIncrement : Increments the build revision number.

-sceneList : A list of scenes to include in the build in CSV format.

-sceneListFile : A json file with a list of scenes to include in the build.

-buildOutput : The target directory you'd like the build to go.

-colorSpace : The ColorSpace settings for the build.

-x86 / -x64 / -ARM / -ARM64 : The target build platform. (Default is x86)

-debug / -release / -master : The target build configuration. (Default is master)
