Remarks

Call Dispose() when you are finished using the OculusApi.OVRNativeBuffer. The Dispose() method leaves the OculusApi.OVRNativeBuffer in an unusable state. After calling Dispose(), you must release all references to the OculusApi.OVRNativeBuffer so the garbage collector can reclaim the memory that the OculusApi.OVRNativeBuffer was occupying.
