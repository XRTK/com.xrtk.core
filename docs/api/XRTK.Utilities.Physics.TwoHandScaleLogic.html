Class TwoHandScaleLogic

Implements a scale logic that will scale an object based on the ratio of the distance between hands.

object_scale = start_object_scale * curr_hand_dist / start_hand_dist

Usage: When a manipulation starts, call Setup. Call Update any time to update the move logic and get a new rotation for the object.
