Enum OculusApi.RawButton

Oculus native button definitions

Namespace: XRTK.Oculus.Plugins
Assembly: XRTK.Oculus.dll
Syntax
[Flags]
public enum RawButton
Remarks

Oculus API only uses RAW definitions. Oculus Asset also uses Virtual mappings, but it's not clear why (Oculus) Raw button mappings that can be used to directly query the state of a controller.
