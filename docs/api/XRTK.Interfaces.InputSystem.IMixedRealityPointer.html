PointerRaycastLayerMasksOverride

The Physics Layers, in prioritized order, that are used to determine the CurrentPointerTarget when raycasting.

Declaration
LayerMask[] PointerRaycastLayerMasksOverride { get; set; }
Property Value
Type Description
LayerMask[]
Remarks

If set, will override the PointerRaycastLayerMasks's default raycasting layer mask array.

Examples

Allow the pointer to hit SR, but first prioritize any (potentially behind SR)

int sr = LayerMask.GetMask("SR");
int nonSR = Physics.DefaultRaycastLayers &amp; ~sr;
IMixedRealityPointer.PointerRaycastLayerMasksOverride = new LayerMask[] { nonSR, sr };
